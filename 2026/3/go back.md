| 认真维护几段深度友谊（真实资产） | 无 | 这个没细做笔记，一个是讲恋爱的，说实话我恋爱谈够了，虽然现在还在谈，也对自己有锻炼，大学我真不想在这种事情上投入时间了（累）友谊可以有，大学的朋友也许确实很铁很有用，到时候再看吧 |
| 学一项可持续练习的身体技能 | 会带来一种具体的进步感“自我效能感”而且出了大学各种设备和时间都要花钱 | 笔者是清华的，但我不能保证我的烂二本能有足够的体育设施，时间还是有的，跑步吧，还有健身，人还是要有更多自我价值的锚定点，不然很容易被单一领域的失败击倒，失败的只是事，人总能从头来过的，不要失去从头再来的勇气啊（跑题） |
| 做一件失败的事 | 比赛/项目/创业 用低成本方式提高自我认知，认识到自己在压力下的挫败状态和自己能够承受的边界 | 哈哈，这一点我自己就悟出来了，高中的失意对于我来说还真是利大于弊，我真的变成熟很多，虽然很坎坷吧，但年轻人的坎坷都是宝藏，从失败中总结和学习就是打开宝箱的钥匙，我会继续给自己找一些失败的机会的 |
| 至少一次独自长途旅行 | 和自己长时间相处，提高自我认知 同时大学生身份景区打折加青旅可穷游，大学毕业同样没钱没时间 | 所以说认识自己很重要啊，旅游也是个认识对象的过程，反过来也可以认识自己，这个和我当初一个人被困白色监狱一个道理，那时候我真的是人生观大改变，那时候的日记也很好品，旅行这一点很多地方都见了，主要是见世面和锻炼，我都想好了，中国周边我再穷也要转一圈的，没人能拦得住我 |
| 找一个真做事的老师跟着做真事 | 主动 | 有关于上海交大生存手册，我对大学的水平和生活一定有了一定的认知，大学教授比三中老师强不到哪里去，还真感谢三中啊，感觉已经上过大学了（欣慰）人际方面，大学还是要放开手脚好好沟通啊，死手快学啊，学和人打交道！ |

 - 总而言之：主动的，有意识的管理自己的时间和注意力。大学四年，不是用来完成，而是用来探索的（一定要狠狠折腾啊！）
